# §FutNoni. نائیخباری

کاتی نائيخباری به ده‌گمه‌ن پێوه‌ندی به relative temporal value هەیە؛ بەڵام دۆخی کردارەکە دەردەبڕێ. دەربڕینی ڕێزمانی نیشاندەری چۆنیەتی کردارێک لە ده‌وری کات و چۆنیەتی پێشکەشکردنی ئه‌و کرداره‌یه. بۆ نموونە، لەوانەیە کردارێک وەکوو پرۆسەیەکی ناتەواو یان بەردەوام بخاتەڕوو. کاتی ئێستای ته‌واو، لە دۆخی کرداری نائيخباری، کردارەکە وەکوو حاڵەت و مەرجێک نیشان دەدات و کاریگەری یان دەرئەنجامی ئەو کردارە کە پێشتر ڕوویداوە جه‌خت ده‌کاته‌وه.

لەگەڵ ئاوەڵناوی کاری:[^126] ئاوەڵناوی کرداری داهاتوو زۆرتر ئاراسته‌ی مه‌به‌ست یان ئاواتێکە و ویست دەردەبڕێت. ئاوەڵناوی کرداری داهاتوو لەگەڵ [ὡς](http://www.perseus.tufts.edu/hopper/morph?l=w%28s&la=greek&can=w%28s0&prior=sunh/lqomen) ئاماژه به ویستی بکه‌ری کار ده‌کات.
