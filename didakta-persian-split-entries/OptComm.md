# §OptComm. امر یا درخواست

افعال تمنایی دوم‌شخص ممکن است برای بیان فرمان، درخواست یا  پیشنهاد به کار روند. این کاربرد تمنایی به وجه امری شبیه است. تمنایی اول‌شخص برای بیان اطاعت از خواسته یا فرمان و یا گرفتن اجازه‌ی انجام یک کار استفاده می‌شود.
